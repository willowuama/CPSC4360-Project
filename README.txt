Instructions for running the program on your local machine:
Make sure you have mongodb and node.js installed on your computer.

Running the Database:
open terminal
run 'mongo'

open another terminal window
run 'mongod'

Running the program:
open terminal
navigate to project folder
run 'node app.js'
